GPA Calculator 
What is this? 
A simple Python program that calculates your semester GPA automatically. 
How it works 
• Enter number of subjects 
• For each subject: input marks (0-100) and credits 
• Program converts marks to grades and calculates GPA 
• Shows final results with feedback 
Grading System 
• 90-100: O (10 points) 
• 80-89: A+ (9 points) 
• 70-79: A (8 points) 
• 60-69: B+ (7 points) 
• 50-59: B (6 points) 
• 45-49: C (5 points) 
• 40-44: P (4 points) 
• Below 40: F (0 points) 
How to use 
1. Save code as gpa_calculator.py 
2. Run: python gpa_calculator.py 
3. Follow on-screen instructions 
Features 
• Easy to use 
• Handles invalid inputs 
• Instant grade display 
• Encouraging messages 
• Works for any subjects 
Requirements: Python 3.x
